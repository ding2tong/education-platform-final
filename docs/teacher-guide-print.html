教學平台使用手冊 - 教師指南

本手冊引導教師建立、管理課程內容以及製作測驗，協助快速完成教育訓練教材上架。

Part 1: 課程管理

課程是所有教學內容的基礎。您可以建立不同的課程，並在其中新增多個教學單元。

課程教材管理畫面

在「課程教材管理」頁面中，教師可以快速掌握所有課程的分類、排序、上架狀態與單元數。

1.1 如何新增課程

  1. 登入後台：使用教師帳號登入系統。
  2. 進入課程管理：在左側管理選單點擊「課程管理」。
  3. 點擊新增課程：在課程列表頁右上角點擊「新增課程」。
  4. 填寫課程資訊：輸入課程標題、描述、分類與上架狀態。
  5. 儲存課程：儲存後會進入課程編輯頁，可開始新增單元。

1.2 如何編輯現有課程

課程編輯與單元管理畫面
課程基本資料：設定課程標題、描述、分類與上架狀態。
課程單元管理：新增單元、編輯單元、刪除單元，或拖曳調整學習順序。
建議先以「草稿」建立課程，確認單元與測驗都完成後，再改為「已上架」。

Part 2: 單元管理

單元是課程的具體教學內容。每個課程可以包含多個單元。

單元 Markdown 編輯與投影片預覽畫面

2.1 在課程中新增單元

  1. 進入課程編輯頁面。
  2. 在「課程單元管理」區塊點擊「新增單元」。
  3. 填寫單元標題、單元描述與 Markdown 內容。
  4. 完成後點擊「儲存單元」。

2.2 編輯單元內容 (Markdown)

平台使用 Markdown 撰寫內容，並整合 Reveal.js 製作投影片。每張投影片之間使用獨立一行 --- 分隔。

# 投影片 1 的標題

這是第一張投影片的內容。

---

# 投影片 2 的標題

- 這是第二張投影片的項目符號。
- 您可以繼續新增內容。

圖片提示詞撰寫規範

圖片提示詞需具體且具畫面感，避免只寫「藥局圖片」、「保健品照片」這類籠統描述。

公式：[風格]，[光線]，[主體] 在 [場景]，[構圖/鏡頭]，[情緒/品牌感/用途]。
範例：溫暖寫實攝影風格，午後柔和自然光，一位藥師在明亮藥局櫃台前整理保健品，近景構圖，乾淨專業、親切可信，適合教育訓練投影片封面。
  • 是否能想像畫面中的主體？
  • 是否說明光線與風格？
  • 是否包含場景與構圖？
  • 是否符合課程用途與品牌感？

嵌入外部內容

可使用 <iframe> HTML 標籤嵌入 YouTube 影片或外部網頁。建議貼入後先查看右側投影片預覽是否正常。

Part 3: 測驗製作

教師可以為整個課程或特定單元建立測驗，以評估學生學習成效。

測驗編輯畫面

3.1 建立課程總測驗或單元小測驗

  1. 課程總測驗：進入課程編輯頁面，找到「測驗內容管理」。
  2. 單元小測驗：進入單元編輯頁面，找到「單元小測驗」。
  3. 點擊「編輯測驗」進入測驗編輯器。

3.2 手動新增與編輯題目

  1. 點擊「新增問題」。
  2. 填寫問題與選項。
  3. 選擇正確答案。
  4. 填寫解釋，讓學生完成測驗後能理解原因。
  5. 重複新增題目並儲存測驗。

3.3 使用 JSON 檔案快速建立測驗

JSON 匯入測驗格式示意

大量題目可使用 JSON 匯入。匯入後請務必再次檢查題目、選項與正確答案。

{
  "title": "課程期末測驗",
  "questions": [
    {
      "question": "Vue 3 的主要響應式 API 是什麼？",
      "options": ["Options API", "Composition API", "React Hooks"],
      "correct": 1,
      "explanation": "Composition API 是 Vue 3 的核心新功能。"
    }
  ]
}

上架前檢查

  • 課程標題與描述清楚。
  • 課程分類正確。
  • 單元順序符合學生學習流程。
  • Markdown 投影片預覽正常。
  • 圖片提示詞具體、可執行。
  • 測驗題目、答案與解釋皆已檢查。
  • 確認無誤後，再將課程狀態改為「已上架」。
