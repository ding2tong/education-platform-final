# 教學平台使用手冊 - 教師指南

歡迎使用本教學平台！本手冊將引導您如何建立、管理課程內容以及製作測驗，幫助您快速上手。

---

## Part 1: 課程管理

課程是所有教學內容的基礎。您可以建立不同的課程，並在其中新增多個教學單元。

### 1.1 如何新增課程

1.  **登入後台**：使用您的教師帳號登入系統。
2.  **進入課程管理**：在左側的管理選單中，點擊「課程管理」。
3.  **點擊「新增課程」**：在課程列表頁面的右上角，您會看到一個「新增課程」的按鈕，點擊它。
4.  **填寫課程資訊**：
    *   **課程標題**：為您的課程取一個清晰、吸引人的標題。
    *   **課程描述**：簡要介紹課程內容、目標學習者等資訊。
    *   **課程分類**：選擇課程所屬的類別（例如：教育訓練、商品教育）。
    *   **上架狀態**：
        *   選擇「草稿」：課程將不會對學生顯示，適合在內容準備階段時使用。
        *   選擇「已上架」：課程將會公開發布，學生可以開始學習。
5.  **儲存課程**：填寫完畢後，點擊頁面底部的「儲存課程」按鈕。儲存後，您將被引導至課程的編輯頁面，可以開始新增單元。

### 1.2 如何編輯現有課程

1.  **進入課程管理**：從管理選單進入「課程管理」頁面。
2.  **選擇課程**：在課程列表中，找到您想修改的課程，點擊右側的「編輯」按鈕。
3.  **修改內容**：您可以修改課程的標題、描述、分類和上架狀態。
4.  **儲存變更**：完成修改後，點擊「儲存課程」按鈕。

## Part 2: 單元管理

單元是課程的具體教學內容。每個課程可以包含多個單元。

### 2.1 在課程中新增單元

1.  **進入課程編輯頁面**：請先依照 `1.2` 的步驟進入您想新增單元的課程編輯頁面。
2.  **點擊「新增單元」**：在「課程單元管理」區塊，點擊「新增單元」按鈕。
3.  **填寫單元資訊**：
    *   **單元標題**：設定單元的標題。
    *   **單元描述**：簡短描述這個單元將教授的內容。
    *   **單元內容 (Markdown)**：這是單元的核心內容。請參閱下一節以了解如何編輯。
4.  **儲存單元**：完成後，點擊「儲存單元」。

### 2.2 編輯單元內容 (使用 Markdown)

本平台使用 Markdown 格式來撰寫豐富的教學內容，並整合了 Reveal.js 簡報工具，讓您可以輕鬆製作投影片。

*   **基本語法**：
    *   `# 標題1`：最大的標題
    *   `## 標題2`：次級標題
    *   `* 項目符號` 或 `- 項目符號`：建立項目清單
    *   `**粗體文字**`：將文字變為粗體
    *   `*斜體文字*`：將文字變為斜體

*   **製作投影片 (Reveal.js)**：
    *   若要將您的內容分割成多張投影片，請在每張投影片之間使用 `---` (三個減號) 分隔。

    *   **範例**：
        ```markdown
        # 投影片 1 的標題

        這是第一張投影片的內容。

        ---

        # 投影片 2 的標題

        - 這是第二張投影片的項目符號。
        - 您可以繼續新增內容。
        ```

*   **嵌入外部內容 (例如 YouTube 影片)**：
    *   您可以直接使用 `<iframe>` HTML 標籤來嵌入影片或其他網頁。
    *   **YouTube 嵌入步驟**：
        1.  在 YouTube 影片頁面，點擊「分享」。
        2.  選擇「嵌入」。
        3.  複製提供的 `<iframe>` 程式碼。
        4.  將程式碼貼到 Markdown 編輯器中即可。

## Part 3: 測驗製作

您可以為整個課程或特定單元建立測驗，以評估學生的學習成效。

### 3.1 建立課程總測驗或單元小測驗

1.  **進入課程或單元編輯頁面**：
    *   **課程總測驗**：進入課程編輯頁面，找到「測驗內容管理」區塊。
    *   **單元小測驗**：進入單元編輯頁面，找到「單元小測驗」區塊。
2.  **點擊「編輯測驗」**：點擊按鈕後，您會進入測驗編輯器。

### 3.2 新增與編輯題目

1.  **點擊「新增問題」**：在測驗編輯頁面，點擊此按鈕以新增一個題目。
2.  **填寫題目內容**：
    *   **問題**：輸入您的題目。
    *   **選項**：平台預設提供四個選項欄位 (A, B, C, D)。請填寫您的選項內容。
    *   **正確答案**：從下拉選單中選擇正確的選項。
3.  **重複新增**：您可以重複以上步驟，新增多個問題。
4.  **調整題目順序**：按住題目左側的拖曳圖示，即可上下拖動以調整順序。
5.  **刪除題目**：點擊題目右側的「刪除」按鈕即可移除該題。
6.  **儲存測驗**：完成所有題目的編輯後，點擊頁面底部的「儲存測驗」按鈕。
